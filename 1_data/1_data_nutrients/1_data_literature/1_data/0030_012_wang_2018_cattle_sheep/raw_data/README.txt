-----------------------------------------------------------
The first row in each file contains variable names with units separated by underscores (e.g. C_mg/g).
The arcsine square root transformation was used for statistical analysis of proportional data.
The encoding is UTF-8 without BOM.
-----------------------------------------------------------

-----------------------------------------------------------
File name: 
  Diet_C_N.CSV
File description:
  C/N ratios of herbivores diet.
-----------------------------------------------------------

-----------------------------------------------------------
File name:
  Diet_Selection.CSV
File description:
  Proportions of each plant species foraged by herbivores.
-----------------------------------------------------------

-----------------------------------------------------------
File name:
  Feces_C_N_Release.CSV
File description:
  Feces C loss (CR), N loss (NR), C release rates (CRR) and N release rates (NRR) during decomposition.
-----------------------------------------------------------

-----------------------------------------------------------
File name:
  Feces_Chem_Fiber.CSV
File description:
  The chemical composition and fiber contents of fresh feces. The data include fecal C, N, hemicellulose, cellulose, lignin (ADL) and lignin/N ratios.
-----------------------------------------------------------

-----------------------------------------------------------
File name:
  Feces_CN_Ratio.CSV
File description:
  Dynamics of fecal C/N ratios during decomposition.
-----------------------------------------------------------

-----------------------------------------------------------
File name:
  Feces_Mass_Moisture.CSV
File description:
  Dynamics of feces remaining ratio (dry matter) and water content during decomposition.
-----------------------------------------------------------

-----------------------------------------------------------
File name:
  Feces_MBC.CSV
File description:
  Dynamics of feces microbial biomass carbon during decomposition.
-----------------------------------------------------------

-----------------------------------------------------------
File name:
  Plant_C_N.CSV
File description:
  The C and N contents of plants foraged by herbivores.
-----------------------------------------------------------

-----------------------------------------------------------
File name:
  Soil.CSV
File description:
  The mineral N contents and water contents of soil under feces bags at day 0 and day 30 during decomposition.
