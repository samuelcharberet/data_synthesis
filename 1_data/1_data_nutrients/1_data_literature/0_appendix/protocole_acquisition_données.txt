--- PROTOCOLE POUR RENTRER LES DONNÉES ---


1° dans le fichier, repérer les données et les encadrer en rouge et mettre des bookmarks si posssible

Vérifier s'il y a des données supplémentaires disponibles

2° Attribuer un code à la référence: sur le modèle 1234_001_authorname_year
	1234: whether nutrient data is provided on 
		food (put a 1, otherwise 0, in first position)
		body (put a 2, otherwise 0, in second position)
		feces (put a 3, otherwise 0, in third position)
		urine (put a 4, otherwise 0, in fourth position)
	001: the article number

Example: An article published in 1996 by Yokoshima with chemical data on food and urine, and this article is the third one in this category:
1004_003_yokoshima_1996

3° Entrer l'article dans le fichier database_papers

4° Créer un dossier dans to_check, avec le meme nom que le pdf. Ce dossier doit contenir:
- un fichier readme contenant auteur année et titre de l'article et une description succinte des données
- le fichier pdf dont le nom est le même que celui du dossier qui le contient
- les données supplémentaires
- le fichier excel dont le nom est obligatoirement sur le modèle 1234_001_authorname_year


5° rentrer les données dans le template. Se réferer à la description de chaque colonne dans le template. 


- 6° rentrer la référence dans zotero (cf explications séparées)
- 7° mettre les tags ("marqueurs") décrivant les données, dans zotero
- 8° déplacer le dossier avec le pdf + xlsx depuis TOTO vers 2_DONE_TO_CHECK, dans la section équivalente


--- ZOTERO ---

1° Si pas déjà fait, Créer un dossier NetSto Data avec les meme sous dossiers que celui de TODO

2° Ajouter une ref
pour ce que j'en comprends, il faut d'abord créer les refs en local dans le dossier perso NetSto

pour ça, cliquer sur le sous dossier de destination de la ref
glisser le pdf dans la partie du milieu
attendre 10s : les métadonnées vont être extraites (auteur, etc)
vérifier les métadonnées (il y a souvent des erreurs)
mettre les tags (marqueurs appropriés)

3° mettre la ref dans la biblio partagée

- glisser la ref depuis le dossier perso data NetSto vers le dossier en ligne, dans le dossier adéquat

(le dossier en ligne est en bas à gauche dans la section bibliothèques de groupe)

- ensuite il faut synchronier le dossier : fleche tournante verte vers la droite, en haut à droite de la fenètre de zotero.
