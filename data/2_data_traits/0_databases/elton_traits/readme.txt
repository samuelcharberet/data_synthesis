EltonTraits

Diet, foraging strata, foraging time, and body size, in particular, determine a large portion of the “Eltonian” niches of species. 
Here we present a global species-level compilation of these key attributes for all 9993 and 
5400 extant bird and mammal species derived from key literature sources. 

https://doi.org/10.6084/m9.figshare.c.3306933.v1

This database can be downloaded using R and the traitdataform package using pulldata("eltontraits").
