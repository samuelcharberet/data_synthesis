données sur les food habits et les physical features de 5735 espèces

- traits_adw.csv : données brutes extraites de la base de données avec le script traits_adw_script

- gbif_id_adw.csv : données gbif sur les espèces de la base adw obtenues avec le script gbif_id_adw

- complete_adw.csv : association de traits_adw et de leur accepted_gbif avec le script gbif_id_adw

- adw_ready_to_merge.csv : harmonisation du data frame pour merger avec elton - script prep_merge_adw

 
NB. chaque espece est associé au gbif id de l'accepted species correspondante
